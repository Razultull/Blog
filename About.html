This site serves as not only a creative outlet for me but also somewhere I can test my newfound knowledge in CSS and java scripting. I intend to become adept at XHTML such that later I may be able to smoothly transition to HTML 5 the upcoming standard. With javascript and some bits of flash( I know its going to be extinct soon) I can hopefully create a website that is clean and attractive and easy to navigate from page to page.




This site and all the files attached were first laid out in Photoshop, then saved as png/jpeg files and then subsequently implemented into HTML and CSS using Dreamweaver. Dreamweaver has allowed me to quickly establish a workflow using Photoshop to create mockups and then achieve those mockups using CSS3.
